# Neonates-Cry-Detection

This project is part of a **Smart Neonatal Incubator Monitoring System**.  
It uses **Deep Learning (CNN)** to detect infant crying from audio signals and displays a **medical alert** on a web interface.

## Features
- Infant cry detection using CNN
- Real-time audio processing
- Flask-based medical alert web interface
- CPU-based execution (GPU supported optionally)

## Technologies Used
- Python 3.10
- TensorFlow / Keras
- Librosa
- Flask
- HTML / CSS
